# TODOS LIST APP WITH VITE + REACT + TS

Basic website that allows users to ADD, DELETE and EDITH items on the website.

## Built With

- REACT
- VITE
- Github.


## Key Features:

- Add, delete and edith an item to the list.
- Login feature
- Hamburger menu.


## Getting Started

### Setup

1- clone Repo.

```sh
   https://github.com/efmomoh/react-todo-app.git
```

### Prerequisites:

**you can run any one of these commands to install npm on your machine:**

- npm i
- npm install
- sudo apt install npm

  **to install globally, just add a -g or --global after the install or i**

- npm i -g
- sudo apt install -g npm

### Install

2- Install npm (node pakage manager)

### Usage

```sh
  use git bash to open in Vs code;
  use npm start to run in web browser
  use  npm install uuid   for universal unique identifier id project
  use npm install react-icons  to install react icons
  use  npm install react-router-dom for router library at root project
```

### Run tests

Use the CLI for launching this project live.

```sh
  run npm run dev  
```
